# Smart Job Advert Analyzer

A web application that analyzes job advert images and extracts key information automatically.

## Features
- OCR text extraction
- Detects application emails
- Detects application links
- Extracts closing dates
- Identifies key requirements
- Generates job summaries

## Tech Stack
- Next.js
- Tesseract.js
- Tailwind CSS

## How to Run
npm install
npm run dev
